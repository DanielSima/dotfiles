# Raspberry Pi 1
These are dotfiles for my raspberry pi #1.

It contains my dotfiles and other configuration files for Home Assistant, Pi-hole and PiVPN.

## Usage
    
download this branch:

	git clone --single-branch --branch raspberry_pi_1 https://github.com/DanielSima/dotfiles.git /home/[USER]/dotfiles/raspberry_pi_1

give execute permission:

	sudo chmod +x /home/[USER]/dotfiles/raspberry_pi_1/raspberry_pi_1.sh

and execute the script:

	/home/[USER]/dotfiles/raspberry_pi_1/raspberry_pi_1.sh
